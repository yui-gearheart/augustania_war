安裝方法：

1.下載遊戲文件（yui_game.py）和數據文件夾（game_data）

2.將yui_game.py第9行的yui_dir = 'game_data/'改為數據文件夾的實際路徑

3.運行yui_game.py



使用方法：

本遊戲分為挑戰BOSS和玩家對戰兩種模式

1.在下方選擇隊伍的1的5位英雄

2.若選擇挑戰BOSS，則在上方選擇要挑戰的BOSS，若選擇玩家對戰，則在上方選擇隊伍2的5位英雄

3.在右方選擇遊戲進行速度（默認為慢速）

4.點擊「開始戰鬥」，即開始戰鬥，雙方輪流使用技能，戰鬥記錄顯示在中央消息框中，隊伍1的戰鬥資訊用綠色顯示，隊伍2的戰鬥資訊用藍色顯示

5.點擊「停止戰鬥」可提前終止戰鬥，並輸出傷害和治療統計結果
