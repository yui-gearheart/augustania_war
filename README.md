安裝方法：
1.下載遊戲文件（yui_game.py）和數據文件夾（game_data）
2.將yui_game.py第9行的yui_dir = 'D:/game_data/'後面的路徑改為數據文件夾的保存位置
3.運行yui_game.py

使用方法：
1.在下方選擇5位出戰的英雄
2.在上方選擇要挑戰的BOSS
3.在右方選擇遊戲進行速度（默認為慢速）
4.點擊「開始戰鬥」，即開始戰鬥，雙方輪流使用技能，戰鬥記錄顯示在中央消息框中
5.點擊「停止戰鬥」可提前終止戰鬥，並輸出傷害和治療統計結果
